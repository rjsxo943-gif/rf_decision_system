# RF 환경 기반 무선 인프라 의사결정 시스템

## 실행 방법
```bash
python main.py --mode collect      # 데이터 수집
python main.py --mode preprocess   # 전처리
python main.py --mode train        # 학습
python main.py --mode evaluate     # 평가
python main.py --mode gui          # 대시보드
python main.py --mode replay       # 발표 리허설
```
